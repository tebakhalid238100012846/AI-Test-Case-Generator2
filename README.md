# AI-Assisted Test Case Generator

This project demonstrates how Artificial Intelligence techniques can assist in automatically generating software test cases.

The system analyzes Python source code and automatically generates unit tests for the functions found in the code.

Technologies used:
- Python
- PyTest
- AST (Abstract Syntax Tree)

Project Workflow:
1. Read the source code from target_code.py
2. Detect functions automatically
3. Generate test cases for each function
4. Execute the tests using PyTest
5. Display the test results

Example Output:

6 passed in 0.02s

This means all generated tests were executed successfully.
